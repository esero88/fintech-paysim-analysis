# PaySim Fraud Analysis

This project analyzes financial transactions using the PaySim dataset.

## Project Steps
- Data ingestion using Kaggle API
- Data exploration and basic analysis
- Fraud pattern identification

## Key Insight
Fraud is only observed in TRANSFER and CASH_OUT transactions.

## Tech Stack
- Python
- Pandas
- Kaggle API

## Note
Dataset is not included due to size. It can be downloaded using Kaggle API.
